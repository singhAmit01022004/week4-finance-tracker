# 💰 Personal Finance Tracker

A comprehensive Python-based command-line application designed to help users manage their daily expenses, track budgets, and visualize spending habits.

## 🚀 Features
* **Expense Management**: Add, view, and search through expenses.
* **Data Persistence**: Automatically saves data to JSON and supports CSV exports.
* **Budget Tracking**: Set a monthly budget and monitor your status.
* **Visual Reports**: Generates text-based charts for category breakdowns.
* **Safety**: Built-in backup and data recovery system.

## 🛠️ Technical Architecture
The project follows a modular structure to ensure clean code and easy maintenance:



* `main.py`: The entry point and user interface.
* `expense.py`: Defines the Expense object and data validation.
* `file_handler.py`: Manages JSON storage, CSV exports, and backups.
* `reports.py`: Handles data analysis and visualization logic.
